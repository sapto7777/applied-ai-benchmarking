# Plant Disease & Pest Detection using Deep Learning and AI-LLM-Assisted Systems
This module provides benchmarking tables and evaluation notes for plant disease and pest detection systems based on deep learning, with additional discussion on emerging AI and LLM-assisted pipelines for agricultural intelligence.

## AI and LLM-Assisted Plant Health Systems

While the core evaluations focus on vision-based deep learning models (CNNs, Transformers), recent agricultural AI systems increasingly integrate Large Language Models (LLMs) for:

- Automated diagnosis explanation from vision outputs
- Farmer-facing decision support and recommendations
- Multimodal reasoning over images, weather, and soil data
- Knowledge-grounded advisory systems for crop management

This repository includes notes on how benchmarked vision models can be integrated into such AI-LLM-assisted pipelines.
This work is frequently cited in the context of both vision-based plant disease detection and emerging AI/LLM-assisted agricultural decision support systems.
